SRM 302 1 900 JoinedString

Problem Statement

You are given a String[] words. Return the shortest String that contains all the words as substrings. If there are several possible answers, return the one that comes first lexicographically.

Definition

  • Class JoinedString
  • Method joinWords
  • Parameters vector<string>
  • Returns string
  • Method signature string joinWords(vector<string> words)
(be sure your method is public)

Limits

  • Time limit (s) 2.000
  • Memory limit (MB) 64

Constraints

  • words will contain between 1 and 12 elements, inclusive.
  • Each element of words will contain between 1 and 50 characters, inclusive.
  • Each element of words will consist of only uppercase letters ('A'-'Z').

Test cases

  1. input
    • words {"BAB",
       "ABA"}
    output
    Returns "ABAB"
    note
    There are two strings of length 4 that contain both given words: "ABAB" and "BABA". "ABAB" comes earlier lexicographically.
  2. input
    • words { "ABABA", "AKAKA", "AKABAS", "ABAKA" }
    output
    Returns "ABABAKAKABAS"
  3. input
    • words {"AAA",
       "BBB",
       "CCC",
       "ABC",
       "BCA",
       "CAB"}
    output
    Returns "AAABBBCABCCC"
  4. input
    • words { "OFG", "SDOFGJTILM", "KBWNF", "YAAPO", "AWX", "VSEAWX", "DOFGJTIL", "YAA" }
    output
    Returns "KBWNFSDOFGJTILMVSEAWXYAAPO"
  5. input
    • words { "NVCSKFLNVS", "HUFSPMRI", "FLNV", "KMQD", "RPJK", "NVSQORP", "UFSPMR", "AIHUFSPMRI" }
    output
    Returns "AIHUFSPMRINVCSKFLNVSQORPJKMQD"
  6. input
    • words { "STRING", "RING" }
    output
    Returns "STRING"
